2. Información que Recopilamos

Este sitio web recopila información a través de:

  • Cookies de Google Translate: Para proporcionar funcionalidad de traducción automática
  • Cookies técnicas: Para el correcto funcionamiento del sitio (modo oscuro, preferencias)
  • Google Sheets: Para actualizar dinámicamente el menú del día (no recopila datos personales)

Este sitio web NO recopila datos personales directamente de los usuarios. No hay formularios de contacto ni sistemas de registro.

3. Finalidad del Tratamiento

Los datos técnicos recopilados se utilizan exclusivamente para:

  • Garantizar el correcto funcionamiento del sitio web
  • Proporcionar funcionalidades de traducción automática
  • Recordar las preferencias del usuario (modo oscuro)
  • Mostrar información actualizada del menú

4. Base Legal

La base legal para el tratamiento de datos técnicos es el interés legítimo en proporcionar un servicio web funcional y mejorar la experiencia del usuario.

5. Destinatarios de los Datos

Los datos técnicos pueden ser compartidos con:

  • Google Translate: Para proporcionar servicios de traducción
  • Google Sheets: Para actualizar el contenido del menú

6. Derechos del Usuario

Los usuarios tienen derecho a:

  • Acceso: Conocer qué datos se están tratando
  • Rectificación: Corregir datos inexactos
  • Supresión: Solicitar la eliminación de datos (borrar cookies del navegador)
  • Oposición: Oponerse al tratamiento de datos
  • Limitación: Solicitar la limitación del tratamiento
  • Portabilidad: Recibir los datos en formato estructurado

Para ejercer estos derechos, puede ponerse en contacto con nosotros a través del teléfono 963 91 78 63.

7. Conservación de Datos

Los datos técnicos (cookies) se conservan durante el tiempo necesario para cumplir con la finalidad para la que fueron recabados. Las cookies pueden eliminarse en cualquier momento desde la configuración del navegador.

8. Seguridad

Café Restaurante L'Express ha adoptado medidas de seguridad técnicas y organizativas apropiadas para proteger los datos contra acceso no autorizado, pérdida o alteración.

9. Modificaciones

Esta política de privacidad puede ser modificada. Se recomienda revisar periódicamente esta página para estar informado de cualquier cambio.

10. Reclamaciones

Si considera que sus derechos han sido vulnerados, puede presentar una reclamación ante la Agencia Española de Protección de Datos (www.aepd.es).
